StressSense+ - Employee Stress Detection System
===============================================

Thank you for installing StressSense+!

WHAT IS STRESSSENSE+?
---------------------
StressSense+ is an employee stress detection system that uses computer vision
and AI to monitor employee stress levels through webcam analysis. The system
runs automatically in the background and sends stress data to a central server
for management review.

HOW IT WORKS
-----------
1. The system captures webcam images at regular intervals (default: every 30 minutes)
2. AI analyzes facial expressions to detect stress levels
3. Only stress data is sent to the server - no images are transmitted
4. Managers can view stress trends through the web dashboard

FIRST TIME SETUP
---------------
1. Launch StressSense+ from the desktop shortcut
2. Enter your Employee ID when prompted
3. Enter a device name (e.g., "Office Computer")
4. The system will register your device and start monitoring

SYSTEM REQUIREMENTS
------------------
- Windows 10 or later
- Webcam (built-in or external)
- Internet connection
- Administrator privileges for installation

PRIVACY & SECURITY
-----------------
- Images are processed locally on your computer
- No images are ever sent to the server
- Only stress analysis results are transmitted
- All data is encrypted in transit
- Device registration ensures data privacy

TROUBLESHOOTING
--------------
If the application doesn't work:
1. Ensure your webcam is connected and functioning
2. Check that you have internet connectivity
3. Verify that the service is running (check Windows Services)
4. Try restarting the application

For technical support, contact your IT department.

UNINSTALLATION
-------------
To uninstall StressSense+:
1. Go to Windows Settings > Apps > Apps & features
2. Find "StressSense+" in the list
3. Click Uninstall
4. Follow the uninstaller prompts

Alternatively, run the uninstaller from the Start Menu.
